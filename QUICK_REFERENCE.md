## ğŸ› Troubleshooting Quick Tips

### Common Issues & Solutions

**"ModuleNotFoundError: pygame"**
```bash
pip install pygame
```

**"No available video device"**
```bash
python3 launch_game.py --mode text  # Use text mode
# Or on Linux with X11:
xvfb-run python3 launch_game.py --mode graphics
```

**Low FPS / Performance Issues**
```bash
python3 profile_graphics.py         # Identify bottlenecks
# Then edit config: Lower resolution or FPS
```

**Can't load save file**
```bash
python3 automation/validate_save_files.py  # Check saves
```

**Import errors**
```bash
python3 diagnose.py                 # Full diagnostic
```

**For more:** See [docs/TROUBLESHOOTING.md](docs/TROUBLESHOOTING.md)

## ğŸ¯ Development Workflow

### Daily Development
1. Check health: `python3 diagnose.py`
2. Make changes
3. Run tests: `python3 test_graphics_system.py`
4. Commit: `git commit -m "message"`

### Adding Features
1. Update roadmap: Edit `docs/ROADMAP.md`
2. Implement feature
3. Add tests to `test_graphics_system.py`
4. Update documentation
5. Profile: `python3 profile_graphics.py`
6. Validate: `python3 automation/validate_*.py`

### Bug Fixing
1. Reproduce bug
2. Run: `python3 diagnose.py`
3. Check: `docs/TROUBLESHOOTING.md`
4. Profile if performance: `python3 profile_graphics.py`
5. Fix and test
6. Add regression test
